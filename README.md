# 한마디 (Hanmadi)

A Korean learning platform built with Next.js.

## Features

- 📚 Vocabulary Notebook
- 🎥 Video Learning
- ✍️ Writing Review
- 🗣️ Speaking Practice

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.
